# Socket/server

API server for Socket.


## Get Started

1. Make sure you have Node.js, NPM and/or Yarn installed.
2. Install all dependencies (use either, not both)

  **Yarn**

  ```
  $ yarn 
  ```
  
  **NPM**
  
  ```
  $ npm i
  ```
3. Run development server

  **Yarn**

  ```
  $ yarn dev
  ```
  
  **NPM**
  
  ```
  $ npm run dev
  ```
